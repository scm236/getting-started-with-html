Web basics
You (and your browser) request to the Internet, which requests to servers (closet computers hosting files)
HTML: “HyperText Markup Language” – base for all web pages
Made up of: text content (what you see), markup (what it looks like)
Can contain references to other documents (images/videos/etc), links to other pages

HTML tags - in all cases, replace parentheses with greater/less than signs to make functional
Bold (b)bold(/b)
Emphasis (em)italicize(/em)
Image tag: (img): (img src=”url” alt=”text”) [nice to include alt just in case url doesn’t work][“void” tag doesn’t need a closing tag]
Break tag (br) makes a line break [another void tag], stack if you want multiple breaks
Paragraph tag (p) content (/p) will render as a paragraph
Difference between break and paragraph tags: former is inline (just text), whereas paragraph makes a block (with height and width)
Span and div both contain content (inline and block, respectively): (span) text (/span), (div) text (/div)

Attributes: (tag ATTR=“value”)contents(/tag)
Anchors make links (a href = www.reddit.com)derp(/a) links to Reddit and displays “derp”
